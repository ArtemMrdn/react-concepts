1) Компоненты - это строительные блоки любого React приложения, и компоненты выстраиваються в четкую иерархическую структуру, у каждого компонента кроме корневого, должен быть 
1 родительский компонент, а также у любого компонента может быть много дочерних компонентов, компоненты можно при этом переиспользовать

2) JSX - это синтаксическая надстройка на JS, и с помощью JSX, используя JS а также синтаксис похожий на HTML и CSS можно описать любой компонент React, но в конечном итоге JSX конвертируется
в JS

3) Свойства - это вводная информация для любого компонента, и свойства передаются от родительского к дочернему компоненту, причем свойства внутри компонента менять крайне не рекомендуется, 
они должны меняться только в родительском компоненте

4) Состояние - если нужна динамика в компонента, т.е динамическая поведение в компоненте непосредственно то необходимо использовать состояние, при этом тот компонент в котором состояние и
создано должен быть единственным компонентом, который имеет право изменять это состояние, 

Кроме этого части свойст и состояние можно передавать другим дочерним компонентам в виде свойств
